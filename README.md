## Graph Expansion Method

Starting from the 47 seed profiles, we scraped GitHub following lists for all users with accounts.
People followed by **multiple** seed awesomers were prioritized. Notable intersections:

- `karpathy` — followed by rasbt + aravindsrinivas
- `lh3` (Heng Li) — followed by crazyhottommy + nh13
- `biobenkj` — followed by rasbt + crazyhottommy
